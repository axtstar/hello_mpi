## install open-mpi

> brew install open-mpi

## build

> cargo build

> cargo build --release


## run

> mpirun -np 2 ./target/release/hello_mpi

use hostfile

> mpirun -hostfile hostfile  -np 4 ./target/release/hello_mpi
